Install the +thingle bookmarklet

First, display the bookmark bar for your browser, which will appear above the URL address bar

Chrome: Click the icon on the browser toolbar and select Bookmarks. Then, select the option to always show bookmarks bar.

Safari: Click View in the toolbar and click show bookmarks bar.

Click and drag this into the bookmark bar     Bookmarklet

Use the +thingle bookmarklet

Create thingles from content anywhere on the web

See something you want to thingle? While on a page, simply click on the +thingle bookmarklet and it will automatically find images for your thingle. Select one, and the add thingle dialog box will appear.
