What is the +thingle bookmarklet

Once installed in your browser, the +thingle bookmarklet lets you grab an image from any website and use it to create a new thingle. When you create thingles from other websites we automatically grab the source link so we can credit the original creator

How to Install the +thingle bookmarklet

First, in case it is not already there, make sure the bookmark bar for your browser is visible. It will appear above the URL address bar:

For Chrome: Click the icon on the browser toolbar and select Bookmarks. Then, select the option to always show bookmarks bar.

For Safari: View > Show Bookmarks Bar

For Firefox: Click View > Toolbars > Bookmarks Toolbar

Click and drag this into the bookmark bar    +thingle

How to Use the +thingle bookmarklet

See something you want to collect as a thingle? While on a page showing a photo of that thing, simply click on the +thingle bookmarklet and it will automatically find all suitable images on the page. Select the one you want, and the add thingle dialog box will appear.
